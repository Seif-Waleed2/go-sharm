Overview

Go Sharm is a smart tourism platform designed to digitally transform the tourism experience in Sharm El-Sheikh by connecting tourists with hotels, activities, restaurants, and local service providers through a unified web and mobile platform.

The platform simplifies tourism discovery, booking, and service management while empowering local businesses with digital exposure.

------------------------------------------------------------------------------------------------>

Problem Statement

Tourists often face:

Difficulty finding trusted tourism services in one place

Fragmented booking experiences across multiple platforms

Limited access to real-time offers and local recommendations

Service providers also lack a centralized digital platform to showcase their services.

------------------------------------------------------------------------------------------------>

Solution

Go Sharm provides:

A centralized tourism marketplace

Easy booking for hotels, trips, and activities

Interactive maps and smart recommendations

Deals, discounts, and real-time updates

Dashboards for tourism service providers

System Architecture

The system follows a 3-Tier Architecture:

Frontend: React Web App

Backend: Node.js + Express REST APIs

Database: PostgreSQL / MySQL

Integrations: Payment Gateway, Maps API, Notification Services

Technology Stack

Frontend: React, BootStrap

Backend: Node.js, Express.js

Database: PostgreSQL

Authentication: JWT

Cloud Hosting: AWS / Azure
